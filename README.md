# Raspberrypi monitor project

泥电连续性综合设计




## Raspberrypi + picamera
树莓派摄像头与流式传输画面

### Installation

```bash
$ sudo apt-get install python-flask
$ sudo pip install picamera
```

### Run it

```bash
$ cd raspberry-camera/
$ python appCam.py
```

![](image.png)

## Frontend 


### Installatio

```bash
$ cd vue-frontend
$ npm install
```
###

```bash
$ npm run dev
```
