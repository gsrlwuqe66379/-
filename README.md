# Raspberrypi monitor project

泥电连续性综合设计




## Raspberrypi + picamera

### Installation

```bash
$ sudo apt-get install python-flask
$ sudo pip install picamera
```

### Run it

```bash
$ cd raspberry-camera/
$ python appCam.py
```
